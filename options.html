Create a Notion Integration

You need a Notion API integration token to connect this extension. Create a private integration in your Notion workspace.

1
Go to notion.so/my-integrations and click "New integration"
2
Give it a name (e.g. "Web Clipper"), select your workspace, set capabilities to Read + Insert content, and save.
3
Copy the "Internal Integration Secret" token (starts with ntn_ or secret_)
4
In any Notion page you want to use as parent: click ⋯ → Connections → Add connection and select your integration.
2
Paste Your Integration Token
Stored locally in Chrome — never sent anywhere except Notion's API.
3
Default Database

Choose a Notion database to clip into by default. You can always change it per-clip in the popup.
